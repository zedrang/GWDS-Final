Coding Worksheets

These are the worksheets that I have completed the coding tasks on each week

The first task was a very basic look at making the base
outline of a website, but it mostly introduced me to using
GitHub pages which is a very useful tool for coding that I
will probably use a lot in the future and helped me make
this site.

The Repository for Sheet 1

In the second and third tasks we looked at some more
basic HTML and CSS elements that for me served as a good
refresher from my Website Development module in college. I
made different styled lists, put in links, used different
fonts and colours, and linked two pages together.

The Repository for Sheet 2 and 3

In the fourth and fifth tasks I learnt some new code in
the form of flexboxes, these are very useful for creating a
responsive website. We also created buttons which I'd never
done before as well as embedding audio and video and using
Google and Adobe fonts.
